# AMI Overview

- AMI = Amazon Machine Image
- AMIs are a customization of an EC2 instance
  - You can add your own software, operating system, monitoring, etc
  - Faster boot/configuration time - all software is pre-packaged
- Bound to a specific region
- EC2 instances can be launched from:
  - Public AMI (Provided by Amazon)
  - Bring your own AMI
  - AWS Marketplace AMI
