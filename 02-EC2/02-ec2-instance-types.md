# EC2 Instance Types

- Different types of EC2 instances are optimized for different use cases
  - [Instance Types](https://aws.amazon.com/ec2/instance-types/)

- General Purpose
  - Used for a most workloads
  - Balance between
    - Compute
    - Memory 
    - Networking

- Compute Optimized
  - Used for compute intensive tasks that require high performance processors
    - Batch processing
    - Media transcodeing
    - High performance web servers
    - HPC
    - Machine learning
    - Gaming servers

- Memory Optimized
  - Workloads that process large datasets in memory
    - High performance databases
    - Distributed web scale cache stores
    - In memory databases
    - Real time processing of unstructured data

- Storage Optimized
  - Used for storage intensive tasks that require high speed read/write access
    - High frequency online transactions
    - Databases
    - Cache for in-memory databases
    - Data warehousing
    - Distributed file systems
