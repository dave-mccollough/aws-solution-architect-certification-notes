# IAM MFA Overview

- Password Policy

  - Strong Passwords - better account security
  - Minimum password length
  - Require specific character types
    - Uppercase
    - Lowercase
    - Numbers
    - Non-alpha characters
  - Allow IAM users the ability to change their own passwords
  - Define password expiration
  - Prevent password reuse

- Multi-factor Authentication (MFA)
  - Authenticator app
  - Security key
  - Hardware fob
